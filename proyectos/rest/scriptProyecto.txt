1. Instalar Docker
https://www.docker.com/products/docker-desktop/

2. Instalar el contenedor con Mysql
docker run --name mysql-container -e MYSQL_ROOT_PASSWORD=xideral2025 -p
  3306:3306 -d mysql:latest

3. Crear usuario y base de datos en mysql
Este script crea el usuario xideral, la base de datos academy_db y otorga todos los privilegios sobre esa base de datos al usuario.

CREATE USER 'xideral'@'%' IDENTIFIED BY 'xideral';
  CREATE DATABASE IF NOT EXISTS academy_db;
  GRANT ALL PRIVILEGES ON academy_db.* TO 'xideral'@'%';
  FLUSH PRIVILEGES;

4. Esta tabla incluye id (clave primaria), nombre, email (único), teléfono y fecha de registro con timestamp automático.

CREATE TABLE Cliente (
      id INT AUTO_INCREMENT PRIMARY KEY,
      nombre VARCHAR(100) NOT NULL,
      email VARCHAR(150) UNIQUE NOT NULL,
      telefono VARCHAR(20),
      fecha_registro TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  );
